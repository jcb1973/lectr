# Lectr — Project Root

Workspace root for the Lectr project. Contains the `CLAUDE.md` project brief
for Claude Code and symlinks to the four component repos for local navigation.

## Components

| Directory  | Description              |
|------------|--------------------------|
| `iOS/`     | iOS app (Swift)          |
| `Android/` | Android app (Kotlin)     |
| `API/`     | API server (Cloudflare Workers) |
| `website/` | lectr.io (Static HTML)   |

Each component is an independent git repo. See `CLAUDE.md` for full project
architecture, privacy principles, and release process.
